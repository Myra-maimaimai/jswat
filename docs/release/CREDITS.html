JSwat: Credits

JSwat is principally developed and maintained by Nathan Fiedler, with contributions from various individuals. As of version 3.0, JSwat owes a great deal to the NetBeans organization. The NetBeans Platform provides the application framework on which JSwat is deployed.

Apache BCEL

JSwat makes use of the Apache BCEL for the byte codes view. The license for this library is included in the bcel.jar file found in the JSwat binary and source distributions.

Artwork

Many of the icons in JSwat are borrowed from NetBeans IDE.
