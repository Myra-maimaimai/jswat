Reporting Bugs

I Found A Bug!

First and foremost, confirm that you really found a new bug. To do this, follow the steps outlined below. This should not take long.

  1. Check the list of known bugs to make sure this has not already been found and reported by someone else.
  2. Are you using the latest version of JSwat? If not, you probably ought to try to reproduce the problem using the latest version. The bug may have been found and fixed already.
  3. Read the JSwat documentation carefully. There isn't much and it will be worth your time.

If after all that it seems like you have found a new bug, please send as much of the following information as you possibly can to the jswat-bugs mailing list.

  • Operating System (e.g. Windows, Linux, Solaris)
  • JDK version (e.g. 1.2.2, 1.3.1, or 1.4.0)
  • JDK brand (e.g. Sun, IBM, Blackdown)
  • JSwat version (hopefully the latest)
  • Your classpath and sourcepath settings
  • What you were trying to do
  • The steps you followed
  • Any pertinent error messages
