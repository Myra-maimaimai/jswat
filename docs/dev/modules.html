Writing New Modules

Writing new modules for JSwat is as easy as writing new modules for the NetBeans Platform. Simply use the NetBeans IDE 5.5 or later to quickly set up a module project. Prior to starting, see the instructions on compiling JSwat to get your build environment set up correctly.

To learn more about using NetBeans IDE module support, see the built-in help, and the web site documentation.

What not to Check In

Try to avoid checking in the following files into the source repository:

  • nbproject/genfiles.properties
  • nbproject/build-impl.xml
  • nbproject/private/

To achieve this, set the svn:ignore property on the nbproject directory like so:

genfiles.properties
build-impl.xml
private
