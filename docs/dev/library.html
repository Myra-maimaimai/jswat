Adding a new Library

When adding a new jar to one of the core Java projects in JSwat, it is necessary to update the project configuration for the nbcore project. The nbcore project is a library wrapper module project, which happens to contain several jar libraries instead of the usual one. See the explanation on the NetBeans wiki for the details. We are using the second method, and have added several class-path-extension elements to the nbcore/nbproject/project.xml file.

Thus, adding a new jar library requires simply following through with the instructions given on the NetBeans wiki, and ensuring that a new class-path-extension element is add to the nbcore/nbproject/project.xml file.
