JSwat Installer

The installer project is found in the installer directory in the JSwat source tree. This is a simple Java application that extracts the JSwat production build to a location of the user's choosing, and makes some minor changes to the extracted files.

Testing

Testing the installer is simple, just invoke the jar target of the installer/build.xml Ant script, then run the installer/dist/installer.jar file. This is most easily done if you open the installer project in NetBeans. The installer has the necessary dummy data to perform a dry-run install, which is sufficient for making sure it works.

Building

Building the installer is achieved by building JSwat for release. See the release document for instructions.
