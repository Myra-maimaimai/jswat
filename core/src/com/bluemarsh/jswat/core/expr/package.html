Defines the evaluator of Java-like expressions. The evaluator can parse and execute most of the typical Java statements. Below are the Java language features this evaluator does not process (it will throw appropriate exceptions in such cases).

  • new - this keyword is not supported.
  • ?: - the ternary operator is not supported.
  • super - this keyword is not supported.
  • Class.this.member - member access of the outer class from inside a static inner class is not supported.

The Evaluator class is the facade to this package. It provides the necessary methods to evaluate an expression and return the result. This should be the only class which clients need to call upon to evaluate expressions.
