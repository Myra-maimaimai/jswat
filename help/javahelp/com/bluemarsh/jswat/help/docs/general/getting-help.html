Getting Help

See Also

The following page describes what to do when you run into a problem that is preventing you from getting work done. Please take a moment and read through this page before sending an email asking for help.

  1. Read the User's Guide in the JSwat installation directory, which explains most of the primary debugger functionality.
  2. Read the help pages on the pertinent topics. The commands also have a help feature, just invoke the help command.
  3. Review the FAQ on the web site to see if this problem has come up before.

To ask questions about JSwat, please send an email to the jswat-discuss@googlegroups.com mailing list. When asking for help, be sure to mention the operating system, JDK version, and the JSwat version that you are using.
