Session Settings

See Also

Session settings are set using the Session Settings dialog box. You can open the dialog box by choosing Session > Settings from the menu bar. Below is a description of the available settings.

Name
The session name is a simple moniker of your choosing. You may enter any name you like, and it does not have to be unique.
Classes
The entries in the Classes list make up the classpath for the debuggee, for those cases where the debuggee is launched from within the debugger. If the session is presently connected to the debuggee, then this list is read-only and reflects the actual classpath.
Sources
The entries in the Sources list make up the sourcepath for the debuggee. See the section below for a more detailed explanation.

Sourcepath

For more information on the sourcepath and how it is used, see the Classpath and Sourcepath topic.
